## Project Overview
This project is a **Library Management System**.  
It includes:
- ER diagram and relational model
- MySQL code to implement the database
- Backend and frontend PHP files for managing the library system

The system helps manage books, members, authors, librarians, and publishers. It tracks borrowed books with issue and due dates.

## Project Structure
- **Backend:** PHP files for login, database connections, and library operations  
- **Database:** `librarysystem.sql` contains tables and schema  
- **Frontend:** HTML/CSS/JS files for interface  
- **Documents:** This project document explains ER diagrams, entities, and MySQL code
